Kial mi fariĝu gastiganto?

{% asvar Question %}Sed mi jam gastigas eĉ sen esti en la Pasporta Servo!{% endasvar %} {% asvar Answer %} Se vi oficiale aperas en Pasporta Servo, ankaŭ novuloj kaj homoj el aliaj anguloj de la planedo povas vin trovi. {% endasvar %} {% include './snippets/faq_item.html' with Question=Question Answer=Answer only %} {% asvar Question %}Sed venos tro da gastoj, se mi estos en la gastiganto-listo!{% endasvar %} {% asvar Answer %} Vi ĉiam rajtas rifuzi tranokto-peton sen klarigi kial. {% endasvar %} {% include './snippets/faq_item.html' with Question=Question Answer=Answer only %} {% asvar Question %}Sed mi ne povas gastigi la tutan jaron, mi ne ofte estas hejme.{% endasvar %} {% asvar Answer %} La gastoj ĉiam devas anonci sin antaŭ ol veni al vi, do vi vidos, ĉu vi tiam estos hejme aŭ ne. {% endasvar %} {% asvar AnswerAlt %} Vi krome povas indiki ĉe via loĝejo periodon kiam vi ne havas eblecon gastigi, se vi anticipe konas la datojn.

Se vi estas gasto…

{% asvar Question %}Ĉu estas limo pri kiom da adresoj aŭ tranoktebloj mi rajtas uzi?{% endasvar %} {% asvar Answer %} Ne, ne ekzistas limo. Ĉiam dependas de la unuopaj gastigantoj, ĉu ili povas vin gastigi aŭ ne. {% endasvar %} {% include './snippets/faq_item.html' with Question=Question Answer=Answer only %} {% asvar Question %}Ĉu mi vere, ĉiam devas antaŭe kontakti la gastigantojn?{% endasvar %} {% asvar Answer %} Kelkaj gastigantoj pretas akcepti gastojn sen antaŭa anonco, sed nur por malmultaj tio estas tute akceptebla.
En la paĝo de loĝloko (kaj en la libreto, apud la adreso) vi ĉiam vidas la kondiĉojn indikitajn de la gastiganto. Se oni petas interkonsenton kontakti pli frue telefone, do faru tion; se retletere, tion.
Bonvolu ĉiam respekti la petitajn limigojn de la gastiganto. {% endasvar %} {% include './snippets/faq_item.html' with Question=Question Answer=Answer only %} {% asvar Question %}Kiom da tempo mi rajtas resti ĉe la gastiganto?{% endasvar %} {% asvar Answer %} Same kiel pri kontaktado, ankaŭ la resto-daŭro estas indikita. Kelkaj akceptas nur por unu nokto; okazas, ke tute ne gravas por la gastiganto kiom longe vi restas. Vi antaŭanoncu kiom da noktoj vi deziras resti, kaj nepre interkonsentu. {% endasvar %} {% include './snippets/faq_item.html' with Question=Question Answer=Answer only %} {% asvar Question %}Kion mi faru se gastiganto rifuzas mian tranokto-peton?{% endasvar %} {% asvar Answer %} Tio foje okazas. Oni ne ĉiam havas tempon kaj la kondiĉojn gastigi. Serĉu alian gastiganton. {% endasvar %} {% include './snippets/faq_item.html' with Question=Question Answer=Answer only %} {% asvar Question %}Ĉu mia gastiganto ankaŭ donos al mi manĝaĵon?{% endasvar %} {% asvar Answer %} Denove, tio estas laŭ interkonsento. La gastiganto rajtas peti repagon por la manĝado, sed la gastiganto tute ne devas proponi manĝojn. Vi, kiel gasto, povas proponi kuiri aŭ prepari kune! {% endasvar %} {% include './snippets/faq_item.html' with Question=Question Answer=Answer only %} {% asvar Question %}Ĉu mia gastiganto helpos min akiri vizon aŭ restadpermeson?{% endasvar %} {% asvar Answer %} Ne. La ideo de Pasporta Servo estas, ke oni helpas pri la tranoktado, en neformala maniero. Gastigantoj ne povas (kaj foje eĉ ne rajtas) respondeci pri vi antaŭ la instancoj de sia lando. Ĉiujn oficialajn paperojn necesajn por laŭleĝa restado en la lando, vi devas aranĝi formale, mem. {% endasvar %} {% include './snippets/faq_item.html' with Question=Question Answer=Answer only %}

Pasporta Servo – libra versio.

{% asvar Question %}Kiel mi sciu, ke mi aperos en la libro?{% endasvar %} {% asvar Answer %} En la paĝo de via loĝejo, sube, “printi en la PS-libreto”: devas esti jes.
Krome, se vi estis ĝis nun en la libreto, kaj vi denove volas aperi en ĝi, vi konfirmu per la retletero, kiun ni sendos al vi. {% endasvar %} {% include './snippets/faq_item.html' with Question=Question Answer=Answer only %} {% asvar Question %}Mi ne plu volas aperi en la libro. En la redakta formularo iru tute malsupren kaj vidu ĉu estas markilo apud “Printu en la PS-libreto”, se jes, forigu ĝin kaj konservu. {% endasvar %} {% asvar AnswerAlt %} Se vi ne scias kiel ŝanĝi vian profilon, skribu al ni. {% endasvar %} {% include './snippets/faq_item.html' with Question=Question Answer=Answer Answer_Alt1=AnswerAlt only %} {% asvar Question %}Mi ankoraŭ neniam estis en la libro, sed nun volas aperi en ĝi. Klaku sur la butono Redakti tie. En la redakta formularo iru tute malsupren kaj certiĝu ke estas markilo apud “Printu en la PS-libreto”, konservu. Ankaŭ certiĝu ke en la privatecaj agordoj la loĝejo estas markita ke ĝi estu videbla en la libreto.
